This project analyzes Netflix's content library to uncover trends and patterns in genre, release year, regional content, and ratings. The project is a data analytics exercise in Python, using libraries like Pandas, Matplotlib, and Seaborn for data cleaning, visualization, and exploration.


Walmart Sales Analysis Source code

This project analyzes Walmart's sales data using SQL to uncover insights on revenue, trends, branch performance, seasonal trends, product performance, customer demographics, and operational efficiency, aiming to optimize operations and improve customer satisfaction levels.

This project provides insights into the Data Science job market, analyzing job descriptions, salary estimates, required skills, and location-based trends. It uses a cleaned dataset of job postings to create meaningful visualizations and uncover trends that can guide professionals in the field.


Fifa 21 Analysis Source code

This project analyzes the FIFA 21 dataset to explore player attributes, market values, and potential growth patterns using Python. The analysis includes data cleaning, feature engineering, exploratory data analysis (EDA), and predictive modeling to extract valuable insights into player performance and value.
